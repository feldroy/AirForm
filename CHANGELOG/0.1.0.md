## AirForm 0.1.0: Staking the claim

AirForm turns Pydantic models into validated HTML forms. Define your data model once, and AirForm handles rendering the form fields and validating submissions. It works standalone or as part of the Air web framework.

This release reserves the PyPI package name and establishes the project scaffold. The vision: you already define your data as Pydantic models, so why not have your forms come from those models too.

```sh
uv add AirForm
```

Built with [Cookiecutter PyPackage](https://github.com/audreyfeldroy/cookiecutter-pypackage).
