# Installation

## Stable release

To install AirForm, run this command in your terminal:

```sh
uv add AirForm
```

Or if you prefer to use `pip`:

```sh
pip install AirForm
```

## From source

The source files for AirForm can be downloaded from the [Github repo](https://github.com/feldroy/AirForm).

You can either clone the public repository:

```sh
git clone https://github.com/feldroy/AirForm
```

Or download the [tarball](https://github.com/feldroy/AirForm/tarball/main):

```sh
curl -OJL https://github.com/feldroy/AirForm/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd AirForm
uv sync
```
